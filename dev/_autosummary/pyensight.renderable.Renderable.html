Renderable#

class pyensight.renderable.Renderable(session: Session, cell_handle: Optional[Any] = None, width: Optional[int] = None, height: Optional[int] = None, temporal: bool = False, aa: int = 1, fps: float = 30.0, num_frames: Optional[int] = None)#

Generate HTML pages for renderable entities

This class provides the underlying HTML remote webpage generation for the Session ‘show()’ method. The approach is to generate the renderable in the EnSight session and make the artifacts available via the websocketserver. The artifacts are then wrapped with simple HTML pages (also served up by websocket server) which can be used to populate iframes, etc.

Args:
session:

The pyensight session to generate the renderables for

Methods

Renderable.browser()

Open a webbrowser page to display the renderable content

Renderable.delete()

Delete all the server resources for this renderable

Renderable.download(dirname)

Download the content files for this renderable

Renderable.update()

Update the visualization and display it

Attributes

Renderable.url

The URL to the renderable content
