start#

DockerLauncher.start() → Session#

Start EnShell by running a local Docker EnSight image.

Once this method starts EnShell, it connects over gRPC to EnShell running in the Docker container. Once connected, EnShell launches a copy of EnSight and WSS in the container. It then creates and binds a Session instance to the created EnSight gRPC connection and returns this instance.

Returns:
object

PyEnSight Session object instance.

Raises:
RuntimeError

Variety of error conditions.
